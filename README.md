# 📊 E-Commerce Customer & Sales Analytics Dashboard

An end-to-end **Data Analytics** project built using **MySQL, SQL, Python, Power BI, and DAX** to analyze the Brazilian Olist E-Commerce dataset. This project demonstrates the complete analytics workflow—from data extraction and cleaning to exploratory data analysis, statistical analysis, customer segmentation, and interactive dashboard development.

## 🚀 Project Overview

This project transforms raw e-commerce data into actionable business insights by integrating SQL, Python, and Power BI. The analysis focuses on customer behavior, sales performance, payment trends, geographic distribution, and product performance to support data-driven business decisions.

## 🎯 Business Objectives

- Analyze sales and revenue trends.
- Identify high-value customers and customer segments.
- Evaluate product category performance.
- Analyze payment methods and order status.
- Discover regional sales trends by state and city.
- Build interactive dashboards for business decision-making.

## 📂 Dataset

- **Dataset:** Brazilian Olist E-Commerce Dataset
- **Records:** 99K+ Orders
- **Transactions:** 370K+ Records
- **Tables:** 8 Relational Datasets

## 🛠️ Tech Stack

- **Database:** MySQL
- **Query Language:** SQL
- **Programming:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Visualization:** Power BI
- **Data Modeling:** DAX
- **Version Control:** Git & GitHub

## 📌 Project Workflow

1. Data Import into MySQL
2. SQL Data Extraction & Analysis
3. Data Cleaning using Python (Pandas)
4. Exploratory Data Analysis (EDA)
5. Statistical Analysis
6. Customer Segmentation
7. Power BI Dashboard Development
8. Business Insights & Recommendations
